# 💫 O mnie

Cześć! Jestem pasjonatem programowania, który uwielbia tworzyć aplikacje webowe i narzędzia ułatwiające codzienne życie. Specjalizuję się w nowoczesnych technologiach frontendowych i backendowych, a do swoich projektów często integruję modele AI takie jak Claude AI czy DeepSeek. Prowadzę własną stronę [l3st.pl](https://l3st.pl), gdzie możesz zobaczyć moje projekty i aktywność.

---

# 🔧 Stack technologiczny

**Frontend:** Next.js, React, Svelte, Tailwind CSS, TanStack Query, HLS.js  
**Backend:** Node.js, Express, Supabase, PostgreSQL, Prisma ORM, SQLite, Node-Media-Server, FFmpeg  
**AI:** Claude AI, DeepSeek  
**API i integracje:** Google Flights, Skyscanner, Booking, Uber, OpenWeather, Pusher, Stripe, JWT  
**Hosting i wdrożenia:** Vercel, Supabase Cloud
