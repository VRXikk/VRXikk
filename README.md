# Cześć, jestem VRXikk 👋

Buduję cyfrowe narzędzia, które zarabiają. Prowadzę [**l3st.pl**](https://l3st.pl) — zespół obsługujący projekt od pomysłu do działającego produktu: strony, aplikacje, integracje, AI i produkcja live. Bez przerzucania między podwykonawcami.

Aktywny na rynku **od 2021 roku**, z ponad **100 zrealizowanymi projektami**.

---

## 🛠️ Co buduję

| Obszar | Co konkretnie |
|---|---|
| **Strony sprzedażowe** | Landing page'e i strony firmowe pod konwersję — hero, dowód społeczny, CTA, SEO, piksele Ads |

| **Aplikacje mobilne** | iOS + Android w React Native, single codebase, push, offline, biometryka, App Store & Google Play |
| **Integracje i automatyzacje** | CRM ↔ Meta / Google Ads / mailing, scrapery, boty, webhooki, synchronizacja kalendarzy |
| **AI centrale telefoniczne** | Wirtualna recepcjonistka 24/7 po polsku — kwalifikacja leadów, rezerwacje, zapis do CRM (~70% taniej niż infolinia) |
| **Automatyzacja social media** | RSS → generacja treści AI → auto-posting na Facebook & Instagram przez Meta Graph API |
| **Sklepy e-commerce** | Stripe / BLIK / Przelewy24, integracje z magazynem i kurierami, pełna analityka ścieżki zakupowej |
| **Produkcja live / OBS** | Zdalna obsługa OBS podczas streamów IRL — sceny, overlaye, bonding 2–4 łącz LTE, moderacja chatu |

## ⚡ Stack technologiczny

**Frontend:** Next.js 14+, React (RSC), React Native, TypeScript, Tailwind CSS, TanStack Query, HLS.js  
**Backend:** Supabase / PostgreSQL, Node.js, Express, Prisma ORM, SQLite, Node-Media-Server, FFmpeg  
**AI:** Claude AI (Anthropic), OpenAI, DeepSeek  
**Płatności i API:** Stripe, BLIK, Przelewy24, Meta Graph API, Google Flights, Skyscanner, Booking, Uber, OpenWeather, Pusher, JWT  
**Telekomunikacja:** VoIP, SIP, Twilio  
**Streaming:** OBS WebSocket  
**Hosting:** Vercel, Supabase Cloud, AWS

## 🚀 Wybrane realizacje

- **TripMind** — aplikacja webowa i mobilna, która AI-em układa plany podróży, rezerwuje loty i hotele
- **Centrala telefoniczna AI** — wirtualna recepcjonistka rozmawiająca po polsku, umawia wizyty i zapisuje leady do CRM 24/7
- **RSS → FB & Instagram** — silnik scrapujący treści, generujący posty i relacje, publikujący przez Meta API bez udziału człowieka
- **StreamTips** — platforma donacji z alertami OBS, widżetami, wypłatami w 24h i integracją Stripe
- **Cold-Center** — strona B2B dla usług chłodniczych: +45% zapytań w 3 miesiące od wdrożenia
- **Obsługa OBS dla IRL** — zdalne prowadzenie produkcji podczas transmisji live dla streamerów Twitch / Kick
